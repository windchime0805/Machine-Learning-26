##  Project: Student Performance Prediction

This project uses **Linear Regression** to predict student exam scores based on:

- Hours Studied  
- Sleep Hours  
- Attendance Percentage  
- Previous Scores  

---

## 📂 Dataset
- File: `student_exam_scores.csv`  
- Contains student behavioral and academic data  

---

## ⚙️ Workflow
1. Data Cleaning  
2. Exploratory Data Analysis (EDA)  
3. Data Visualization  
4. Model Training (Linear Regression)  
5. Model Evaluation  

---

## 📈 Result
The model predicts exam scores and shows a positive relationship between study habits and performance.

---

## 🛠️ Tech Stack

- Matplotlib / Seaborn

## 🌸 Project: Iris Flower Classification

This project uses **Logistic Regression** to classify iris flowers into different species based on their physical characteristics.

### 📊 Dataset
- Built-in dataset from Scikit-learn  
- Features:
  - Sepal Length  
  - Sepal Width  
  - Petal Length  
  - Petal Width  
- Target:
  - Flower Species (Setosa, Versicolor, Virginica)

---

### ⚙️ Workflow
1. Data Loading  
2. Exploratory Data Analysis (EDA)  
3. Data Visualization (Pairplot)  
4. Model Training (Logistic Regression)  
5. Model Evaluation  

---

### 📈 Result
The model successfully classifies iris flowers with high accuracy (~95%+), demonstrating clear separability between classes.
